# sync-players

Synchronizes play, pause and seeking of players.

## Usage

1. Pause both videos
2. Start program: `python main.py`
3. Align the videos to be correctly synched
4. Mark both video players as _sync_ in the program
5. Unpause one of the videos

## Requirements

* playerctl
* Python 3
* python-gobject
* Python libraries:
    - textual
    - rich
